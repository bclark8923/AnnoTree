Chrome Extension Instructions

The AnnoTree Chrome Plugin is an in browser toolbar that will allow you to draw or write directly onto your web browser, select your tree (project) and send these "leafs" back to our central collaboration platform without ever leaving your page. In order to install the SDK, follow the instructions below:

  Chrome Extension v0.8
  • 1 Download the extension above. If you are currently using Chrome, Chrome will throw an error but the extension file (AnnoTree.crx) will be in your downloads folder.
  • 2 Go to chrome://extensions in your Chrome browser.
  • 3 Open your downloads folder and drag the file AnnoTree.crx into the chrome://extensions page to install it.
  • 4 Open the AnnoTree Chrome extension by clicking on the AnnoTree icon on any webpage or local file (please note that the AnnoTree Chrome extension does not work on the chrome://extensions page - browse to a webpage or open a local file to use it) and log in.
  • 5 After logging in, the AnnoTree Chrome extension will open on towards the top right of the current web page you are on. Double click the AnnoTree icon to engage feedback mode.
  • 6 Once in feedback mode, use the pen to annotate on the webpage or use text to include notes on top of the web page. Click on the tree icon to select a tree to send your feedback to. Click on the send button once you are done with your feedback to send it to this platform.
