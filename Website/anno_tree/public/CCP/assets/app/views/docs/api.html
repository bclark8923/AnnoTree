Download API V0.1

Currently we only support iOS API integration. To install the API

  • Download the API and unzip the package
  • Add the Framework to your "Link with Binary Libraries"
  • Add the Bundle to your "Copy Bundle Resources"

  • Place this line of code in your AppDelegate

    [[AnnoTree sharedInstance] loadAnnoTree:UIInterfaceOrientationMaskAll withTree:@"Your API Key"];
  • Create a Tree and get the API key from the modify menu
