AnnoTree SDK Integration Instructions

The AnnoTree SDK (currently iOS only) is an in app toolbar that will allow you to draw or write directly onto your application and send these "leafs" back to our central collaboration platform without ever leaving your app. In order to install the SDK, follow the instructions below:

iOS API v0.1
Sample App
  • 1 Download the SDK and unzip the package.
  • 2 Add the Framework to your "Link with Binary Libraries"
  • 3 Add the Bundle to your "Copy Bundle Resources"

  • 4 Place this line of code in your AppDelegate

    [[AnnoTree sharedInstance] loadAnnoTree:@"Your SDK key here"];
    • SDK is only for UIKit based apps at the momennt
    • The SDK key is unique to each tree (an application)
    • The SDK key is found in your tree dashboard located to the left
