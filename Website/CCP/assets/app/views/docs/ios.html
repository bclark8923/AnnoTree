The AnnoTree SDK is an in app toolbar that will allow you to draw or write directly onto your application and send these "leafs" back to our central collaboration platform without ever leaving your app. In order to install the SDK, follow the instructions below:

  1. Download the API and unzip the package.
    • iOS API v0.1
  2. Add the Framework to your "Link with Binary Libraries"
  3. Add the Bundle to your "Copy Bundle Resources"

  4. Place this line of code in your AppDelegate

    [[AnnoTree sharedInstance] loadAnnoTree:UIOrientation withTree:@"{{treeInfo.token}}"];
    Note: UIOrientation will be the orientation your app supports i.e. UIInterfaceOrientationMaskAll
  5. Create a Tree and get the API key from the modify menu
