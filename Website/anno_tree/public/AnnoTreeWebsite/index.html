Collaboration Anywhere

We live in a mobile world where developers and testers often work across the globe rather than down the hall. There are many tools that exist to help business create applications and collaborate on projects, but no streamlined solution that brings together developers, designers, testers, and owners on the same platform.

We aim to be that solution. We wanted the ability to collaborate on all levels of application development, across all devices, at any time, on the same platform. With this vision, we created AnnoTree â€“ a mobile-first platform delivering a centralized location for development teams to collaborate on and streamline the creation process.

The Product

  • The AnnoTree SDK allows anyone to create annotations by drawing or writing directly onto a mobile application indicating a bug or design change.
  • Once satisfied, the user can send these annotations (i.e. "Leaf"s) off to our Central Collaboration Platform.
  • Here, team members can organize, view, and collaborate on these Leaves across multiple devices.
